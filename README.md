# Cognex Service/Unit-Based Revenue Model

This Excel model forecasts Cognex (CGNX) revenue, margins, and net income based on units sold and service assumptions.  

It features a scenario switch (Bull / Base / Bear) so you can instantly see how different assumptions impact results.

## Features
- 5-year unit-based forecast  
- Automatic calculation of gross profit, SG&A, R&D, EBIT, and net income  
- Scenario-driven: Bull / Base / Bear  
- Clean tabs: `Historical`, `Assumptions`, `Forecast`

## Requirements
- Microsoft Excel or Google Sheets
